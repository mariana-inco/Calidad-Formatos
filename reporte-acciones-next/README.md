# Reporte de Acciones

Proyecto Next.js independiente para el formato web **Reporte de Acciones**.

## Ejecutar

```bash
npm install
npm run dev
```

Luego abre `http://localhost:3000`.

## Comportamiento

- Fuente cambia según el tipo de hallazgo.
- El botón `Agregar` inserta acciones en una tabla dinámica.
- Cada acción se puede editar o eliminar.
- El botón `Guardar Reporte` valida los campos principales y muestra el JSON como array en consola.
